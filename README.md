####　ネスト
##### 親のresourcesで指定したコントローラ名に、子のresourcesで指定したコントローラ名が続くURLが生成されるのが確認できます。
このような親子関係を、「ネストする」と言います。
上記のようなネストしたURLを作成することでparams[:post_image_id]でPostImageのidが取得できるようになります。
